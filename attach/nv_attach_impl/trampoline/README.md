# trampoline for PTX code

- `default_trampoline.cu` is a template program, where the built-in trampoline for maps opertions comes from
- `default_trampoline.ptx` is generated by `nvcc` (`make` target) and embedded into `../trampoline_ptx.h`
- Default architecture in `Makefile` is `sm_75` (CUDA 13.x no longer supports `sm_61`)
